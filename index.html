Cyber Coliseum

The wars of the future will be fought in space, by tiny robots.

The Cyber Coliseum hosts battles to the death between two JavaScript-programmed robots. The robots can move and shoot projectiles at each other. If a robot absorbs 10 direct hits, the robot is destroyed.

Program your robot by declaring a "think" function that accepts two arguments:

  • "self" (which represents your robot's internal state)
  • "environment" (which represents the environment)
The interfaces of these objects are as follows:
  • self
    • x: number (immutable)
    • y: number (immutable)
    • radius: number (immutable)
    • moveDirection: number (in radians)
    • shootDirection: number (in radians)
    • move: boolean (true if robot should move in the direction of moveDirection
    • shoot: boolean (true if the robot should fire in the direction of shootDirection
  • environment
    • bounds
      • xMin: -10
      • xMax: 10
      • yMin: -10
      • yMax: 10
    • enemies[] (array, currently always length 1)
      • x: number
      • y: number
      • radius: number
    • enemyProjectiles[] (array)
      • x: number
      • y: number
      • angle: number
      • speed: number (distance per tick)
